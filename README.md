# Student-Result-Management-System

---

## Description
This project implements a **Student Result Management System** using **searching**, **hashing**, and **sorting algorithms** in Python.  
It manages student records efficiently and demonstrates how data structures optimize retrieval and sorting operations.

---

## Features
1. **Sequential & Binary Search** for roll number lookup  
2. **Hash Table** with collision handling for record storage  
3. **Heap Sort** and **Quick Sort** for sorting marks  
4. **Radix Sort** for sorting roll numbers  
5. **Performance comparison** of sorting algorithms  

---

## Data Structures Used
| Technique | Purpose |
|------------|----------|
| Sequential & Binary Search | Roll number lookup |
| Hash Table | Efficient record storage |
| Heap Sort | Sort by marks |
| Quick Sort | Sort by marks efficiently |
| Radix Sort | Sort by roll numbers |

---

## Implementation Details
1. **Hash Table:** Used to store and retrieve student records.  
2. **Sequential & Binary Search:** Used for roll number lookups.  
3. **Heap Sort & Quick Sort:** Used for sorting student marks.  
4. **Radix Sort:** Used for roll number sorting.  
5. **Comparison:** Performance observed across datasets.
